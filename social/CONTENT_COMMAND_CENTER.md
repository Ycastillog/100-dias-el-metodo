# Centro de Mando de Contenido - 100 Dias: El Metodo

Objetivo inmediato: conseguir 10 interesados reales, 3 pagos Alpha y 3 personas ejecutando Dia 1.

## Regla nueva

El contenido no existe para verse bonito.

Existe para convencer a alguien de que:

- esta cansado de improvisar;
- necesita una regla simple;
- puede probar el Metodo en pequeno;
- debe comentar o entrar hoy.

Cada pieza debe tener 4 partes:

1. Dolor real: que problema siente ahora.
2. Mini paso a paso: que puede hacer hoy.
3. Mecanismo: que regla del Metodo lo resuelve.
4. Prueba visual: contador, checklist, ruta, cupos o captura legible.
5. Venta final: el Alpha como continuacion natural.
6. CTA: una palabra o accion clara.

Antes de publicar, responder:

- Que objecion rompe?
- Que imagen prueba que esto es real?
- Que palabra debe comentar?

## Oferta que se comunica

Grupo Alpha de 14 dias.

- Cupos: 10
- Entrada: USD 9
- Resultado medible: Pago -> Acceso -> Dia 0 -> Dia 1 -> Dia 7
- Promesa seria: recuperar control con estructura diaria

No prometer ingresos, cambios magicos ni resultados garantizados.

## Posicionamiento

100 Dias: El Metodo no se vende como motivacion.

Se vende como una prueba corta de ejecucion:

- Dia 0: decides.
- Dia 1: ejecutas.
- Dia 7: vemos si volviste.

La frase base:

```text
No te falta motivacion. Te falta estructura para volver.
```

## Rutina diaria

### Manana

1. Publicar 1 Reel/TikTok.
2. Subir 1 story con encuesta o pregunta.
3. Revisar comentarios con palabras clave: CONTROL, DIA1, VOLVER.

### Tarde

1. Responder DMs.
2. Enviar link de la landing.
3. Registrar interesados en el tracker.

### Noche

1. Publicar o reciclar una pieza secundaria: caption, carrusel o post largo.
2. Medir el dia.
3. Ajustar hook si no hubo conversacion.

## Orden de publicacion semanal

Lunes: problema y dolor.

Martes: estructura y Metodo.

Miercoles: Dia 1 y accion concreta.

Jueves: volver despues de fallar.

Viernes: invitacion Alpha.

Sabado: historia personal o caso de uso.

Domingo: resumen, prueba social y llamada a entrar.

## Reglas de ajuste

Si no hay comentarios:

- El hook esta suave.
- Usar una frase mas directa.
- Ejemplo: "No te falta motivacion. Te falta estructura para volver."

Si hay comentarios pero no DMs:

- CTA poco claro.
- Pedir una palabra exacta.

Si hay DMs pero no registros:

- Falta confianza.
- Enviar la ruta Alpha: Pago -> Acceso -> Dia 0 -> Dia 1 -> Dia 7.

Si hay registros pero no pagos:

- Falta link de pago o confirmacion.
- Activar PayPal/Stripe o confirmar manualmente.

## Prioridad real

No crear mas modulos del Metodo.

Primero validar:

- 10 interesados.
- 3 pagos.
- 3 Dia 1 completados.
- 1 Dia 7 completado.
