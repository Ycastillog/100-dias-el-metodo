# Sistema de Prompts para Redes - 100 Dias

## Objetivo

Usar IA como asistente de produccion de contenido, sin perder la voz del Metodo.

La regla:

```text
No vender fantasia.
No prometer dinero facil.
No sonar como vendedor agresivo.
Vender estructura, ejecucion y regreso al marco.
```

## Datos base del producto

```text
Producto: 100 Dias: El Metodo
Oferta inicial: Grupo Alpha
Precio Alpha: USD 9
Duracion Alpha: 14 dias
Cupos: 10 personas
Promesa seria: recuperar control mediante estructura diaria
Formula: Control + Fortaleza + Direccion
CTA principal: Comenta CONTROL y te envio el acceso Alpha.
Audiencia: personas que empiezan y abandonan, viven reaccionando, procrastinan, quieren facturar pero no sostienen estructura.
```

## 1. Generador de ideas de productos/contenido

Usar cuando haga falta crear angulos nuevos para Reels, posts o carruseles.

```text
Quiero crear contenido para vender un producto digital en Instagram, TikTok y Facebook.

Mi nicho es dominio personal, estructura diaria, disciplina practica y personas que quieren dejar de improvisar.

Mi producto se llama 100 Dias: El Metodo.

Mi publico objetivo son personas que empiezan muchas cosas y abandonan, viven reaccionando al telefono, al cansancio o al animo, y quieren recuperar control para ejecutar mejor.

Su mayor problema es que no les falta deseo, les falta estructura para volver al marco cuando fallan.

Dame 10 ideas de contenido vendible para esta audiencia, ordenadas por facilidad de produccion y potencial de conversion.

Para cada idea dime:
- titulo
- hook de primera linea
- formato recomendado: Reel, carrusel, story o post largo
- dolor que toca
- CTA recomendado
- por que podria convertir en comentarios CONTROL

Se especifico. No me des consejos genericos.
No prometas resultados financieros garantizados.
```

## 2. Constructor de estructura de producto

Usar para mejorar la oferta sin agregar modulos nuevos antes del Alpha.

```text
Quiero mejorar la presentacion de un producto digital llamado 100 Dias: El Metodo.

Ayuda a personas que empiezan y abandonan a recuperar control con una estructura diaria basada en Control + Fortaleza + Direccion.

Ahora mismo la oferta vendible es:
Grupo Alpha de 14 dias, 10 personas, USD 9.

Creame una estructura clara de oferta que incluya:
1. Un nombre comercial atractivo para el Alpha.
2. Un subtitulo que explique la transformacion sin exagerar.
3. Los 5 elementos que recibe el participante.
4. El resultado rapido que debe conseguir en los primeros 10 minutos.
5. Como explicar que no es un PDF ni un curso tradicional.
6. Un mensaje corto para justificar el precio de USD 9.

Haz que se sienta serio, accionable y premium, pero sin hype falso.
```

## 4. Maquina de carruseles para Instagram

Usar para crear carruseles de 7 a 9 slides.

```text
Crea un carrusel de 9 diapositivas para Instagram que venda 100 Dias: El Metodo sin que parezca un anuncio.

Producto:
100 Dias: El Metodo.

Audiencia:
Personas que empiezan y abandonan, se distraen, viven reaccionando y quieren estructura para ejecutar.

Resultado:
Recuperar control personal mediante Dia 0, Dia 1, seguimiento y regreso al marco.

Estructura:
Diapositiva 1: gancho fuerte usando una verdad incomoda.
Diapositivas 2 a 6: contenido de valor puro ensenando una sola idea que el Metodo ayuda a resolver.
Diapositiva 7: mito que el Metodo desmiente.
Diapositiva 8: historia corta o prueba de transformacion.
Diapositiva 9: llamada a la accion suave para comentar CONTROL y recibir acceso Alpha.

Tono:
Cercano, autentico y conversacional.
Nada de lenguaje corporativo.
Nada de promesas irreales.

Incluye tambien el caption completo.
```

## 5. Escritor de captions que venden

Usar para posts, Reels y carruseles.

```text
Escribe un caption para Instagram que venda 100 Dias: El Metodo sin que suene como una venta.

Empieza con un gancho de una sola linea que detenga el scroll.

Despues cuenta una historia personal breve sobre por que existe este Metodo:
personas que quieren cambiar, consumen contenido, empiezan con fuerza, pero abandonan cuando falla el animo.

Luego menciona 3 cosas que la persona podra hacer despues de entrar:
1. completar Dia 0
2. ejecutar Dia 1
3. medir si vuelve al marco hasta Dia 7

Anade una linea de prueba social honesta:
Estoy buscando 10 personas reales para validar el Alpha.

Termina con una llamada a la accion suave:
Comenta CONTROL y te envio el acceso Alpha.

Manten el texto por debajo de 300 palabras.
Tono: genuino, cercano y firme.
Sin palabras de moda.
Sin exageraciones.
```

## 6. Guion que vende a traves de historias

Usar para Reels de 45 a 60 segundos.

```text
Escribeme un guion de 60 segundos para un Reel de Instagram que venda 100 Dias: El Metodo a traves de una historia.

Sigue esta estructura:
1. Empieza con un problema que mi audiencia enfrenta todos los dias.
2. Presentame y cuenta el momento en que entendi que la motivacion no bastaba.
3. Revela la solucion que descubri: estructura diaria y regreso al marco.
4. Comparte un resultado especifico, sin prometer milagros: ejecutar Dia 0, Dia 1 y observar Dia 7.
5. Termina con una llamada a la accion para comentar CONTROL.

Nada de ventas agresivas.
Nada de hype falso.
Escribelo como hablaria una persona real.
Incluye sugerencias de texto en pantalla para cada parte del video.
```

### Guion base

```text
Hook:
Si empiezas con fuerza y abandonas cuando se rompe el dia, esto es para ti.

Historia:
Durante mucho tiempo pense que el problema era falta de motivacion.
Guardaba contenido, hacia planes, empezaba otra vez...
pero cuando llegaba el cansancio, volvia al mismo patron.

Cambio:
Lo que cambio no fue una frase bonita.
Fue tener una estructura simple:
Dia 0 para decidir,
Dia 1 para ejecutar,
y una forma de volver al marco cuando fallas.

Puente:
Eso es 100 Dias: El Metodo.
No es un PDF suelto.
Es un sistema para dejar de improvisar.

CTA:
Estoy buscando 10 personas para el Grupo Alpha.
Comenta CONTROL y te envio el acceso.
```

## 7. Generador de stories diarias

Usar para mantener presencia sin producir mucho.

```text
Dame 7 stories para Instagram para vender el Grupo Alpha de 100 Dias: El Metodo durante una semana.

Cada story debe incluir:
- texto corto en pantalla
- tipo de sticker: encuesta, pregunta, slider o link
- objetivo: dolor, confianza, conversion o seguimiento
- CTA

No uses frases motivacionales vacias.
Hazlo directo, humano y accionable.
```

## 8. Regla de uso semanal

Cada semana producir:

```text
3 Reels con historia
2 carruseles educativos
2 captions largos
7 stories
1 post de cierre o cupos
```

Medir:

```text
Retencion
Comentarios CONTROL
DMs
Clics
Registros
Pagos
Dia 0
Dia 1
Dia 7
```

Si no hay comentarios o DMs, cambiar el hook.
Si hay comentarios pero no registros, cambiar el CTA.
Si hay registros pero no pagos, revisar precio, confianza y link de pago.
