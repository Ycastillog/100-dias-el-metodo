# Biblioteca Afiliada - 100 Dias

## Objetivo

Monetizar alrededor del Metodo sin convertir la marca en una tienda.

El producto principal sigue siendo:

```text
100 Dias: El Metodo
```

Los libros son apoyo, autoridad y monetizacion secundaria.

## Regla

Nunca decir:

```text
Compra estos libros para cambiar tu vida.
```

Decir:

```text
Estos recursos complementan Control, Fortaleza y Direccion.
```

## Aviso requerido

Usar en la web y publicaciones:

```text
Algunos enlaces pueden ser afiliados. Como asociado, puedo recibir ingresos por compras calificadas.
```

## Como configurar enlaces

Pegar los enlaces reales en:

```text
assets/affiliate-links.js
```

Ejemplo:

```js
window.AFFILIATE_LINKS = {
  meditaciones: "https://amazon.com/...",
  manual_epicteto: "https://amazon.com/...",
};
```
