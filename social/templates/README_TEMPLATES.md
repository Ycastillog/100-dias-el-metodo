# Plantillas Visuales

Estas plantillas son base para crear imagenes en Canva, Photoshop, CapCut o cualquier editor.

## Archivos

- `reel_cover_template.svg`: portada vertical 1080x1920.
- `story_poll_template.svg`: story vertical 1080x1920.
- `carousel_cover_template.svg`: portada carrusel 1080x1350.
- `facebook_post_template.svg`: imagen horizontal 1200x630.
- `progress_proof_template.svg`: prueba visual vertical con Dia 1/100, checklist y CTA.
- `alpha_offer_template.svg`: oferta horizontal con cupos, precio y ruta medible.

## Previews listos

Los PNG estan en `previews/`.

- `progress_proof_template.png`
- `alpha_offer_template.png`

## Uso

1. Abrir la plantilla.
2. Cambiar el texto principal.
3. Exportar como PNG.
4. Usar el checklist en `../upload_checklist.md`.

## Regla de conversion

Antes de exportar, confirmar que la imagen responde una pregunta:

- Que dolor activa?
- Que regla muestra?
- Que prueba ensena?
- Que accion pide?

Si solo se ve estetica, todavia no esta lista.

## Tipografia sugerida

Si el editor permite cambiar fuente:

- Titulo: Archivo Black, Montserrat ExtraBold o Anton.
- Texto secundario: Inter, Montserrat o Arial.
