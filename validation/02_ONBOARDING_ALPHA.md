# Onboarding Grupo Alpha

## Mensaje de bienvenida

Bienvenido al Grupo Alpha de 100 DIAS: EL METODO.

Durante 14 dias vas a probar el marco inicial del Metodo.

La formula es:

CONTROL + FORTALEZA + DIRECCION = DOMINIO PERSONAL.

No buscamos perfeccion.

Buscamos comportamiento real.

## Reglas del Alpha

1. Completa Dia 0 antes de iniciar.
2. Registra cada dia como Completado, Parcial o Perdido.
3. Si fallas, continuas al dia siguiente.
4. No maquilles el resultado.
5. Responde el check-in del Dia 7.
6. Responde el cierre del Dia 14.

## Entregables que recibe

Acceso inicial al Metodo:

- Metodo Oficial
- Marco Fundacional
- Programa Dia a Dia
- Guia de Implementacion Personal
- Sistema de Avance

## Dia 0

Responder:

1. Que patron quieres romper?
2. Que quieres recuperar?
3. Cual sera tu regla minima diaria?
4. Que excusa ya no vas a alimentar?

## Primer mensaje despues de Dia 0

Tu objetivo no es sentirte listo.

Tu objetivo es volver al marco.

Manana empieza Dia 1.

## Mensaje Dia 1

Hoy empieza el Metodo.

Hazlo simple:

1. Lee el principio del dia.
2. Ejecuta una accion concreta.
3. Registra el resultado.

No busques intensidad.

Busca continuidad.
